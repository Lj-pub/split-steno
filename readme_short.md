왼쪽 초성, 아래 중성(모음), 오른 종성(받침) 

## 1. 모아치기

'ㄱ' 'ㅏ'를 동시에 눌렀다 떼면 '가'가 입력  
'ㄴ' 'ㅓ' 받침 'ㄱ' 동시 '넉' 입력

초성 = 'ㅇ'  
초성 생략  
'ㅏ' 누르면 '아'

쌍자음  
바로 아래 키랑 같이   
'ㅂ' 'ㄱ' 누르면 'ㅃ  
'ㅈ' 'ㄴ' 누르면 'ㅉ'

모음  
'ㅘ', 'ㅝ'는 'ㅗ' 'ㅏ', 'ㅜ' 'ㅓ  
'ㅛ', 'ㅑ'는 'ㅗ' 'ㅡ', 'ㅏ' 'ㅡ'   
'ㅒ'는 'ㅏ' 'ㅓ'  
'ㅖ'는 'ㅏ' 'ㅓ' 'ㅣ'  

오른쪽 ㅋ 받침 

## 2. 약어

자주 쓰는 단어, 글자 매크로

'.' = 받침 'ㅇ', 'ㅁ'  
'있다.' = 'ㅅ' 'ㅁ' 받침 'ㅂ'
